# 🎉 NEXA Events

**Turning Ideas Into Extraordinary Experiences.**

A modern, premium, and fully responsive Event Organizer company website designed to showcase professional event management services with an elegant user experience and engaging animations. This project highlights a contemporary company profile suitable for corporate agencies, wedding organizers, concert promoters, and large-scale event planners.

## ✨ Features

* 🎨 Modern Minimalist UI/UX
* 📱 Fully Responsive Design
* ✨ Glassmorphism Components
* 🎬 Smooth Scroll & Interactive Animations
* 🖼️ Hero Slider
* 🏢 About Company Section
* 💼 Services Showcase
* 📊 Company Statistics
* 📸 Event Portfolio Gallery
* ⭐ Client Testimonials
* ❓ FAQ Section
* 📞 Contact Form
* 📍 Google Maps Integration
* 🚀 Optimized Performance
* 🌙 Premium Visual Experience
